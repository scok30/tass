# TASS - Official PyTorch Implementation
![](./thumbnail.jpg)

### [CVPR 2024] Task-Adaptive Saliency Guidance for Exemplar-free Class Incremental Learning

[Paper](https://arxiv.org/abs/2212.08251)
## Usage

* Training on CIFAR-100 dataset:

```
$ python main.py
```
